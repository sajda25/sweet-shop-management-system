# Screenshots Directory

## How to Add Screenshots

1. **Run the application**:

   - Backend: `cd backend && npm run dev` (runs on http://localhost:4000)
   - Frontend: `cd frontend && npm run dev` (runs on http://localhost:5173)

2. **Capture these views**:

   - `login.png` - Login/Register page
   - `shop-user.png` - Shop page (logged in as regular user)
   - `admin-dashboard.png` - Admin dashboard with CRUD controls
   - `footer.png` - Close-up of the footer component

3. **Save screenshots** in this directory with the exact filenames above

4. **Recommended tool**: Windows Snipping Tool (Win + Shift + S) or any screenshot utility
